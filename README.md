# Teste de 'Minhas Finanças'

## Casos de Testes
Seguem as análises e testes realizados na aplicação 'Minhas Finanças' que rodou em localhost. Os testes unitários estão dispostos na Planilha compartilhada em 'https://docs.google.com/spreadsheets/d/1s11KoRPhzryzxtuMBNZ_m7CRlWP14aiLD8Lnq_Ov8Ao/edit?usp=sharing'.
Os testes de integração e o End-to-End foram realizados com automação Cypress presentes neste repositório.

## Considerações Iniciais
Devido à minha experiência ainda em desenvolvimento com automação de testes, não consegui atender totalmente ao requisito proposto com o Playwright. Ainda assim, decidi prosseguir com a tarefa utilizando Cypress para demonstrar minha capacidade analítica como QA, identificando cenários relevantes, riscos e comportamentos do sistema e evoluir minha prática em automação de testes.
